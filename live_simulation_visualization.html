Dataset: NASA PROMISE KC1 Target: defects (true/false) Inputs: LOC + McCabe + Halstead + Branch Count

Project Purpose

Predict defect‑prone code modules using static software metrics. This enables smarter testing, focused code review, and better maintenance decisions.

Feature Categories

LOC (Lines of Code) McCabe Complexity Halstead Metrics Branch Count
